beat saber


beat saber made by beat games is a rhythm game that is similar to osu!.
the game is played with a light saber and the player must hit the correct notes in time with the music.

vr headsets


there are alot of different vr headsets that can be used with beat saber.
some of the most popular ones are the meta quest 2 or the oculus rift s.

if you prefer an headset which is for a variety of games.
some are the meta quest 2 or the valve index.
but please note that the valve index is a bit more expensive than the other two.

Mods

for mods there are some things you can do.
you can easily just install mods into your steam version/oculus version with modassistent.
or find other mods in the beat saber discord server.

to play old versions of the game there are 2 ways to do so.
you can play through bslegacylauncher on github

or you can use bsmanager. one of the best launchers for beatsaber. this is the github

for more information you can always ask in the beat saber discord server.
